## 選題方法論：施育廷框架的系統化應用

這份選題報告的收斂歷程，採用的是 **施育廷教授提出的選題框架**——從問題發現、文獻對話、根因分析到創意碰撞的結構化流程。

這個框架的核心價值在於：

- **避免過早聚焦**：還沒看清楚文獻地形就急著定題，容易掉進「自以為創新」的陷阱。
- **避免過度發散**：沒有方法論牽引的發散，最後只會產出一堆彼此無關的點子，缺乏深度。

簡單來說，它幫我在「發散」與「收斂」之間找到可操作的節奏，確保選題同時具備**理論前沿性**與**實務相關性**。

&gt; 📎 詳細的收斂歷程（三階段：學術對話 → 根因分析 → 創意碰撞）與 SCAMPER 奔馳法的跨域組合，請見完整報告 `index.html` 中的「選題收斂歷程與方法論創新」章節。
